# AI Fitness Trainer 🏋️‍♂️

An AI-powered fitness trainer built using Python and computer vision techniques for real-time exercise posture tracking and motion analysis.

## 🚀 Features
- Real-time pose detection
- Exercise posture analysis
- Voice control support
- Exercise repetition tracking
- AI-based movement monitoring

## 🧠 Exercises Included
- Squats
- Lunges
- Cobra Pose
- High Knees
- Arm Rotation
- Tree Pose
- Standing Knee to Elbow Crunches

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
